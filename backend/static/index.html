React frontend is ready to be built.

This FastAPI server will automatically serve the new Vite app from frontend/dist once it exists. Right now you are seeing this fallback page because the frontend bundle has not been generated yet.

  1. Install Node.js 18+ on this machine.
  2. Run cd frontend.
  3. Run npm install.
  4. Run npm run dev for local development or npm run build to let FastAPI serve it.

API docs are still available at /docs, and the app endpoints live under /api, for example /api/upload and /api/chat.
