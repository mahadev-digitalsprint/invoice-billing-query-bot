Document Parser Assistant

The API is running. Available endpoints:

  • GET /docs — Interactive API docs (Swagger UI)
  • GET /health — Health check & dashboard stats
  • GET /files — List indexed PDF files
  • POST /upload — Upload and index PDF files
  • POST /chat — Ask a question about indexed documents
  • GET /history/{session_id} — Retrieve chat history
  • DELETE /history/{session_id} — Clear chat history
